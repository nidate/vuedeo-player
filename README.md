# vuedeo-player

A Video player written in Elecron and Vuejs.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints  files

```
npm run lint
```

### Lints and fixes files

```
npm run pretty
```
